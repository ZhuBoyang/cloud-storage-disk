# 煮雨云存储

## 项目介绍

一款开源的云存储（网盘）系统，支持个人存储或多人存储。

## 功能列表

① 注册（使用邮箱进行注册）

② 登录

③ 上传文件

④ 复制文件

⑤ 移动文件

⑥ 新建文件夹

⑦ 下载文件

⑧ 删除文件

## 运行环境

- `NodeJS`（可以启动`Vue3`的版本即可）

- `JDK1.8`

## 项目技术栈

- `Spring Boot 2.7.6`

- `MyBatis Spring Boot 2.3.0`

- `Fluent MyBatis 1.9.9`

- `mail 1.4.7`

- `Hutool 5.8.15`

- `Redis 6`

- `MySQL 8`

- `Vue 3.2.45`

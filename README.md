# 煮雨云存储

## 项目介绍

一款开源的云存储（网盘）系统，支持个人存储或多人存储。

## 功能列表

- 注册（使用邮箱进行注册）、登录

- 文件上传

- 文件复制、移动

- 新建文件夹

- 文件下载

- 删除文件

## 运行环境

- `NodeJS`（可以启动`Vue3`的版本即可）

- `JDK1.8`

## 项目技术栈

- `Spring Boot 2.7.6`

- `MyBatis Spring Boot 2.3.0`

- `Fluent MyBatis 1.9.9`

- `mail 1.4.7`

- `Hutool 5.8.15`

- `Redis 6`

- `MySQL 8`

- `Vue 3.2.45`
